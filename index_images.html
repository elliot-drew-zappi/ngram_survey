Exploring New PhraseCloud approaches - Survey

The DS team has been looking at different ways to generate wordclouds from verbatim data that show not just single words, but interesting phrases as well. We think that phrases may provide more insightful and useful clouds for clients, and they are a common feature in other wordcloud tools on the web.

The idea is that one of these methods could become a replacement/evolution of the current approach.

The data used here are phrases generated from 20 verbatim datasets - 10 surveys, both likes and dislikes question answers.

You will see two phraseclouds produced by two different methods from the same set of comments. The size of the phrase indicates its relative importance/prevalence in the document vs the others.

We are asking you to compare the paired sets of phrases shown, and decide for each pair which set of phrases you would consider to be more useful/appropriate/valuable to a client if they were shown as a wordcloud on the website?

Click the button (left, right) corresponding to the set of phrases you prefer, or click same if there is no obvious winner.

The maximum number of phrases shown is 20 - sometimes the number will be fewer due to cutoffs/thresholds built into some methods.

After you give an answer, new phrases will be shown for you to compare.

You will need to do a minimum of 20 comparisons before the download button is enabled - of course, you can do more than 20 if you want (and we would encourage that if you have the time and patience).

The order will be random. Once you are done, click the "Click to download results as json" button to get a file containing your results - please send that file to Elliot Drew on Slack.

Alternatively - see phrases as a list.

Note! if you leave the page without downloading the results file, your progress will be lost. So make sure you download the file!
